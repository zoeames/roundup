## Roundup

### About
A gender based HTML5 Canvas game built with Phaser.io.

### Features
- [x] Single Level Game
- [x] Player's can be a square peg or a round hole
- [x] Girl and Boy colors
- [x] User fires arrows and jumps to platforms
- [x] Sounds
